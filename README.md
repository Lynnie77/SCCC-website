# sacred_connections_church_V3
Website for Sacred Connections Community Church.  A place for community help & Food Pantry.

👀 Watch Site Being Built As Changes Happen: https://lynnie77.github.io/SCCC-website/
👀 or it could be viewed at (https://www.sacredconnections810.com/)

🗺️ Site Map:
    ✅ index.html Is Main Landing Page

    ✅ styles.css Is Your Styling Page It Gives Your Site Decorations (in css folder)

    ✅ app.js Is Script For Functions to Make The Scheduler, Count Down etc.. (in js folder)

    ✅ images_logo Folder has variations of your logo

    ✅ images_flyers Folder has flyers for events or sponsorships etc..

💡 To Make The Site Yours Fully U Need The Code,
     1. Download ZIP Folder From Green Code Button @ https://github.com/GeoffStengel/sacred_connections_church_V1, 
     2. Make A Repository On Github In Your Account (Create Account If U Dont Have)
     3. Unzip & Extract Those Files & Folders From Step 1 & Upload to Github

🗲  Make Site Live:
     1. Create Github Pages To Make Site Live & Link Your Domain After
     2. In Repo U Just Created Go To Settings & On Sidebar Click "Pages"
     3. Select Deploy From Branch Choose "Master" Or "Main" & SAVE
🎉   4. Congratulations! (URL is Above give a sec or refresh browser)

🔥 How To Link Godaddy Domain To A Github Pages Site:
     1. DNS Point Godaddy Domain to Github Pages Tutorial: https://youtu.be/dbgEWWBvIxY 🛎️ 
     2. Domain Important - at Section 5 You'll Need These Numbers in Godaddy Manage DNS: https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site/managing-a-custom-domain-for-your-github-pages-site#configuring-an-apex-domain 🛎️
